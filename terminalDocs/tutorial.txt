// This is the Terminal bash/zsh Docs for me!

// How to enter a directory
1. cd <name of dir>

// How to make a directory
1. Cd to dir you want to house the new dir
2. mkdir <dir name>

// How to reverse back to previous directory
1. cd .. // Remember the space between the dots and the cd

// How to create a file in a directory
1. touch <name of file plus extension i.e. .txt>

// How to find what directory path you are in
1. pwd //Stands for print directory
